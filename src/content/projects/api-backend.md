# RESTful API Backend

A production-ready REST API with comprehensive features for modern web applications.

## Features

- JWT authentication
- Role-based access control
- Rate limiting
- API documentation with Swagger
- Database migrations
- Docker containerization

## Tech Stack

- Node.js & Express
- PostgreSQL
- Redis for caching
- Docker & Docker Compose
