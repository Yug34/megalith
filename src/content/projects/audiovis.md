# Audiovis

A Windows XP-like audio visualizer that Works with audio you upload from a file.

## Features

- Upload and visualize any .mp3 file.
- Control the visualization parameters to change patterns.
- Rust+WASM processing.

## Tech Stack

- React
- Zustand
- Rust
- WebAssembly
- Tailwind + ShadCN
